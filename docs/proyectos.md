\# Documentación Instalación Servidor FTP



\## Instalación

Se ha instalado vsftpd en Debian (IP: 192.168.1.42).



\## Configuración

Se editó /etc/vsftpd.conf añadiendo:

\- write\_enable=YES

\- allow\_writeable\_chroot=YES



\## Pruebas

Conexión realizada con usuario 'lean-alumno' usando FileZilla.
